Welcome to the PassportEDU API

This repository is intended to provide insight into the internal workings of our API, so that developers can better understand how to leverage and reuse our technology stack to help global students find global schools.

Feel free to copy or borrow our work. PassportEDU is the open source collaborative solution to finding the right school for the right student and the right student for the right school.

This API

This API uses HATEOAS (Hypermedia as the Engine of Application State) on top of a REST using JSON Hyper Schema, NodeJS, Express, and MongoDB. This repository contains a client written with AngularJS which automagically adapts to changes in the schema and API. https://github.com/bpanahij/passportedu_schema/tree/master/client
